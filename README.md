## üéØ Objective & Key Metrics

The primary goal is to pinpoint bottlenecks and optimize **Wait Times** and **Admission Management** for better resource utilization.

* **Main File:** `Hospital Visit Report.pbix` (Download to access the source data and model.)
* **Key Metrics:** Average Wait Time per Department, Bed/Clinic Occupancy Rate, Visit Volume, and Personnel Efficiency.
